# Rescue Management System

Rescue Management System is a software engineering project designed to provide fast and organized emergency support during accidents, fires, floods, and other disasters.

## Main Features

- User Registration
- Login System
- Emergency Report
- Live Location Tracking
- Rescue Team Assignment
- Admin Dashboard
- Communication System
- Panic SOS Button
- Resource Management
- Hospital Support
- Reports and Analytics
- Volunteer Support System
- AI Risk Prediction
- Drone Rescue Support System
- Language Selection
- Feedback and Rating

## Project Purpose

The main goal of this project is to reduce emergency response time and improve coordination between users, rescue teams, hospitals, and administrators.
